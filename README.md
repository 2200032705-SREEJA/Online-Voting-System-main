# 🗳️ Online Voting System

A secure and scalable Online Voting System built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The platform enables voters to register, authenticate securely, cast votes online, and view election results in a user-friendly interface.

## 🚀 Features

* Secure User Registration & Authentication
* JWT-Based Login System
* Online Vote Casting
* Election & Candidate Management
* Real-Time Result Tracking
* Responsive User Interface
* MongoDB Database Integration
* Admin Dashboard
* Duplicate Vote Prevention
* Role-Based Access Control

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/2200032705-SREEJA/Online-Voting-System-main.git
cd Online-Voting-System-main
```

### Install Dependencies

```bash
npm install
```

### Start Application

```bash
npm start
```

### Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
